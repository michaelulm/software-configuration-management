# Faculty Calculator #

The faculty (n!) of any number is the quotient of all numbers from 1 to n, e.g. 4! = 1 x 2 x 3 x 4

| ----  | :---: |
| 0 	| 1 |
| 1 	| 1 |
| 2 	| 2 |
| 3 	| 6 |
| 4 	| 24 |
| 5 	| 120 |
| 6 	| 720 |
| 7 	| 5.040 |
| 8 	| 40.320 |
| 9 	| 362.880 |
| 10    | 3.628.800 |
